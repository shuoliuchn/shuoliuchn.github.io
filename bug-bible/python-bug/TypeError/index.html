unhashable type: ‘dict’

错误4.1

报错信息：

错误代码：

错误原因：

字典的键需要是可哈希的数据类型。字典是可变数据类型，不可哈希，所以报错。

解决方法：

不用字典和其他不可哈希的数据作为字典的键

附可哈希的数据类型：

  1. 整型
  2. 字符串
  3. 布尔值
  4. 元组

不可哈希的数据类型：

  1. 列表
  2. 字典

‘int’ object is not callable

错误9.1

报错信息：

错误代码：

错误原因：

for循环的本质是将可迭代对象迭代赋值给变量。因为程序自上而下运行，一旦变量名与函数名相同，处在下面的变量会覆盖函数名。当后来想要调用函数时，变量inner表示的不是函数inner的内存地址，而是9的内存地址。9是整数，当然是无法被调用的。

解决方法：

for循环使用新的变量名，不要与其他变量和函数相同。

‘NoneType’ object is not callable

错误10.1

报错信息：

错误代码：

错误原因：

写闭包时return闭包函数的应该是外层函数，而不是内存函数返回自己。return的缩进应该和内层函数的def对齐，而不是在内层函数中。

解决方法：

将return inner()的缩进向外调节。
