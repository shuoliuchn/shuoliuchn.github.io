附，python中变量的命名规范：

  1. 数字、字母和下划线组成

  2. 不能以数字开头

  3. 禁止使用python中的关键字

  4. 变量名要具有可描述性

  5. 变量名要区分大小写

  6. 不能使用中文和拼音

  7. 推荐写法：

    1. 驼峰体
    2. 下划线
