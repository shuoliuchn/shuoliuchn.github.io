pip 更换为国内源

发表于 2020-03-02 分类于 Linux 阅读次数：
由于某些原因，访问国外网站的速度会很慢。而官方的 pip 源恰好就在国外。如果想要加快下载速度，可以考虑换源。

有时候，使用 pip 安装 Python 第三方包会很慢。这是因为官方 pip 服务器设置在国外，有很高的网络延迟。

为了解决这个状况，有些大的公司或高校建立起自己的 pip 镜像源。我们从这些国内的镜像源下载第三方包，速度会快很多。
