和虚拟机一样，docker 的网络也可以有多种设置模式。

网络分类

和虚拟机一样，docker 的网络也可以有多种设置模式。

单机中各个容器之间的网络可以有：

  • Bridge Network：相当于 VMware 的桥接，默认跑这个

  • Host Network：类似于 VM 的 NAT

  • None Network：没网

多机组成集群，每台计算机的 docker 容易之间的网络可以是：

  • Overlay Network：集群网络
