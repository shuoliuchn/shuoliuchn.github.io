docker run 的选项

  • -t 表示容器启动后会进入其 命令行终端

  • -i 表示以 交互模式 运行容器

  • --name 表示设置容器的名称

  • -d 创建一个守护式容器在后台运行（这样创建容器后不会自动登录容器，如果只加 -i -t 两个参数，创建后就会自动进去容器）

  • -p 指定端口映射，格式为：主机（宿主）端口:容器端口

  • -e 变量名="变量值" 设置环境变量

  • --network=host 指定容器的网络连接类型

  • --restart=always 在 docker 重启的时候，是否要自动重启容器

  • -v 把 docker 外接的目录和指定容器内部的目录进行映射，共享文件
