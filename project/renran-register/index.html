给 redis 操作设置事务

在 redis 中，如果出现连续操作多条命令， 那么这些命令，我们可以把他们设置成一个整体，采用redis提供的事务来让它们一起执行。执行过程中保证数据的原子性，要么一起成功，要么一起失败。

所以在注册的验证码功能中，我们就可以使用事务来完成验证码的记录工作。
