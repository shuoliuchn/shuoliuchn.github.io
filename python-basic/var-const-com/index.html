变量名（其他 Python 中的标识符如函数名、类名、模块名等也要遵循这个规则）命名规范：

  1. 变量名只能由数字、字母和下划线组成
  2. 不能以数字开头
  3. 禁止使用python中的关键字
  4. 变量名要具有可描述性
  5. 变量名要区分大小写
  6. 不能使用中文和拼音
  7. 推荐写法：
    • 驼峰体
    • 下划线（官方推荐）

注释

注释的作用是给一些晦涩难懂的代码进行标注或解释。被注释的代码不会被执行。

Python 中的注释分为两种：

  • 单行注释（当行注释）：用 # 开头表示
  • 多行注释：用三对 “ ” 或 ‘ ’ 包裹，可以换行
